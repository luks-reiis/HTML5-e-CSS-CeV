TAGs em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá ser utilizada em cada caso.

  • Páragrafo
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas numeradas
  • Listas com dermacadores
  • Itens de listas
  • Links ou âncoras
  • Endereços
  • Textos em destaque (negrito)
  • Textos em ênfase (itálico)
  • Linhas horizontais
  • Quebras de linhas
